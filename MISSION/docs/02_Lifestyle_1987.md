# 1987年上海生活百态调研 (Lifestyle 1987)

## 1. 交通：拥挤的巨龙 (Transport)
- **巨龙车**：上海街头的主力是两节车厢连接的“巨龙车”（铰接式公交车）。
    - **拥挤程度**：极度拥挤，被称为“推屁股”上车。早晚高峰时，车门经常关不上，需要售票员在下面推。
    - **票价**：普通线路 5分、1角。
- **陆家嘴轮渡事故 (重要背景)**：
    - **1987年12月10日**（小说时间线之后几天），陆家嘴轮渡站发生了震惊全国的踩踏事故，造成严重伤亡。
    - **小说伏笔**：小说发生在11月30日-12月2日，此时大雾天气频发，轮渡经常停航，导致过江难上加难。这可以作为主角出行受阻或感受到城市压抑氛围的背景。
- **出租车**：极少，主要是“上海牌”或进口的“菲亚特”，价格昂贵，普通人几乎不坐。

## 2. 居住：蜗居时代 (Housing)
- **人均面积**：上海市区人均居住面积仅 **6-7 平方米**。
- **石库门与弄堂**：
    - **“七十二家房客”**：典型的居住形态。一个门牌号里住着十几户人家。
    - **煤卫合用**：早晨最壮观的景象是“倒马桶”和在公共水斗刷牙洗脸。
    - **做饭**：煤球炉通常放在走廊里，一到饭点，整个楼道烟熏火燎。
- **住房分配**：房子是单位分的，结婚要排队等房，是年轻人最大的焦虑来源。

## 3. 时尚与娱乐 (Fashion & Entertainment)
- **着装变化**：
    - 街头不再是清一色的蓝灰蚂蚁装。年轻人开始穿**喇叭裤**、**牛仔裤**，戴**蛤蟆镜**。
    - 女性开始穿**红裙子**（受电影《街上流行红裙子》影响），烫大波浪。
    - **西装**：男士穿西装开始流行，但往往不剪袖标，且搭配布鞋或旅游鞋。
- **肯德基 (KFC)**：
    - **1987年11月12日**，中国第一家肯德基在北京前门开业。
    - **上海反应**：虽然上海店要到1989年才开（外滩东风饭店），但北京开业的消息通过报纸和电视传到上海，成为时髦的话题。吃一顿“肯德基”被视为高消费和接轨国际的象征。
- **迪斯科 (Disco)**：
    - 此时的舞厅开始流行，播放着《荷东》、《猛士》等的士高音乐。
