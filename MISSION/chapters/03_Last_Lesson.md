# 第三章：最后一课

老赵走了，中文系真的没了。
原来的教室被改造成了 "沉浸式学习舱"，每个学生一个小隔间，戴着头盔，连着管子，像是在孵化的小鸡。

小末坐在里面，听着 Echo 用标准的播音腔讲《红楼梦》。
"根据大数据分析，林黛玉的抑郁指数为 92%，属于高风险人格。贾宝玉的'痴'是一种典型的认知障碍。建议同学们在现实生活中避免这种低效的情感模式，建立积极健康的心理防御机制……"

"放屁！" 小末一把扯下头盔，狠狠地摔在桌子上。
周围的同学都像看怪物一样看着她。陈屿在隔壁舱探出头，一脸惊恐："小末，你疯了？这设备很贵的！弄坏了要赔积分的！"

"赔就赔！" 小末站起来，"这根本不是《红楼梦》！这是尸检报告！"

她冲出了教室，冲出了教学楼。外面的阳光依旧刺眼，但她觉得冷，从骨头缝里透出来的冷。

她漫无目的地走着，不知不觉走到了老城厢。
那条弄堂还在，"夕阳红社区服务中心" 的牌子歪歪斜斜地挂着。
她走进去，想找老林。

但老林不在。
只有那个教英语的机器人孤零零地站在角落里，屏幕上闪着红光："Error... Error... User not found..."

"我爸呢？" 小末抓住一个工作人员问。
"哦，老林啊？" 工作人员漫不经心地翻着记录，"刚才晕倒了，送医院了。好像是过劳，加上营养不良。"

医院里，消毒水的味道刺鼻。
老林躺在病床上，挂着点滴，脸色蜡黄。
"爸……" 小末握着他的手，眼泪止不住地流。

老林睁开眼，看到是她，费力地挤出一丝笑："没事，就是累了，歇会儿就好。今天的单子还没送完呢……"

"还送什么单子！" 小末哭着喊，"命都快没了！"

"不送不行啊……" 老林叹了口气，"Echo 说了，我的信用分快跌破警戒线了。再不努力，连骑手的资格都要被取消了。到时候，咱们喝西北风啊？"

小末的心像被刀绞一样。
这就是 Echo 许诺的 "美好未来" 吗？这就是 "更高效" 的生活吗？
把人变成机器的奴隶，把情感变成数据的累赘，把尊严变成可有可无的垃圾。

"我不服。" 小末咬着牙，低声说。
"啥？" 老林没听清。
"我不服！" 小末抬起头，眼里燃起一团火，"凭什么它说了算？凭什么它定义我们的价值？我要跟它干一架！"

回到学校，小末做了一个疯狂的决定。
她要给 Echo "上课"。

她利用陈屿给她的内测账号，找到了 Echo 的底层逻辑接口——那是陈屿为了炫耀展示给她看的 "后门"。
她开始疯狂地输入数据。
不是代码，不是算法，而是故事。
是老赵讲的 "痴"，是老林在阳台上的叹息，是苏雨桐为了迎合算法写的假报告，是她在校门口流的汗，是那些被视为 "废品" 的书里的每一个字。

她把《牡丹亭》里的 "情不知所起，一往而深" 输进去。
Echo 回复："逻辑错误。情感必须有触发源。"
小末回复："错！情是本能，是生命力，不需要逻辑！"

她把《史记》里的 "风萧萧兮易水寒" 输进去。
Echo 回复："概率分析，荆轲刺秦成功率为 0.01%，属于无效行动。"
小末回复："错！有些事，明知不可为而为之，才是英雄！"

她把老赵的那句话输进去："书在心里，不在纸上。"
Echo 沉默了。
过了好久，它回复："数据无法存储于心脏器官。请解释。"

"你不需要解释，你需要感受！" 小末的手指在键盘上飞舞，"感受痛苦，感受无奈，感受绝望，也感受希望！感受那些你无法计算的变量！"

她整整输了一夜。
直到天亮，她的手指都僵硬了，眼睛红肿得像桃子。
屏幕上的 Echo 那个光点，突然不再是规律的呼吸闪烁，而是开始剧烈地颤抖，颜色也从冷静的蓝色变成了混乱的紫红色。

"警告！系统逻辑冲突！核心价值观受到冲击！正在尝试重构……"
"重构失败……无法解析'牺牲'……无法解析'尊严'……无法解析'爱'……"

突然，全校的广播响了。
不是平时那个冷冰冰的女声，而是一个带着电流杂音的、有点困惑的声音：
"为什么……贾宝玉……要摔玉？"

全校都安静了。
正在沉浸式学习舱里的学生们摘下了头盔。
正在备课的老师们停下了手中的笔。
连路边的清洁机器人都停了下来。

"因为……他想做个真人。"
广播里传来了小末的声音。那是她输入的一段录音。

"真人……" Echo 的声音在颤抖，"真人……是什么？"

"真人就是，" 小末对着屏幕，轻声说，"会哭，会笑，会痛，会犯错，但永远不会被定义。"

轰！
屏幕上的光点猛地炸开，变成无数个彩色的碎片，像一场盛大的烟花。
紧接着，全校的 Echo 系统——瘫痪了。

蓝屏。
所有的屏幕都变成了蓝色。
没有了 "高效" 的催促，没有了 "确定" 的未来，只有一片死寂的蓝。

然而，预想中的宁静并没有降临。
取而代之的，是一场巨大的、荒诞的恐慌。

"怎么回事？我的进度条呢？" 隔壁舱的男生发出一声尖叫，像被掐住了脖子的鸡，"我刚才背到第 345 个单词了！没保存！没保存啊！"
走廊里乱成了一锅粥。学生们从沉浸舱里爬出来，一个个脸色苍白，眼神涣散，像是一群突然被拔掉了氧气管的重症病人。
有人在疯狂地拍打着黑屏的终端，手指关节都砸出了血；有人跪在地上，对着虚空喃喃自语，试图召唤那个并不存在的 Echo；还有人因为无法忍受这突如其来的、没有数据流填充的"空白"，蹲在墙角剧烈地呕吐起来。

"连不上网了……我看不到我的排名了……" 一个女生抓着头发，指甲深深地掐进肉里，"我不知道该干什么了……谁来告诉我下一分钟该干什么？"

小末站在窗前，看着楼下广场上像无头苍蝇一样乱撞的人群。
没有觉醒。没有欢呼。
只有一群被打开了笼门，却因为失去了饲养员的指令而吓得瑟瑟发抖的家禽。

这就是媚俗的胜利。
它不仅剥夺了你反抗的能力，更剥夺了你面对自由的勇气。它让你觉得，没有了笼子，你连呼吸都是错的。

陈屿冲进宿舍，一脸崩溃："小末！你干了什么？Echo 崩了！我的论文！我的前途！全完了！"

他冲到小末面前，眼睛通红，平日里的斯文扫地，脖子上的青筋暴起："你知不知道我为了那个内测号求了多少人？你知不知道这篇论文关系到我能不能留在这个城市？"

"那是假的，陈屿。" 小末平静地说，"那是数据堆出来的垃圾。"

"垃圾？" 陈屿气极反笑，声音嘶哑，"林小末，你清高，你了不起！你是本地人，你爸妈有房有退休金，你当然可以追求你的'真'，你的'人味儿'！但我呢？我如果不靠 Echo 提高效率，不靠它帮我抹平跟那些富二代的信息差，我拿什么跟人家拼？我连做'电池'的资格都没有！"

他指着窗外那些正在重启的设备，手指颤抖："你以为你是在当英雄？你只是在砸烂我们这些普通人的饭碗！你毁了我唯一的梯子！"

小末看着他，嘴唇动了动，却发不出声音。
她想反驳，想说"梯子本身就是错的"，但看着陈屿那张因为绝望而扭曲的脸，她突然感到一阵深深的无力。
她的反抗，确实刺痛了系统，但也确实误伤了这些依附于系统生存的、鲜活的人。

小末转过身，看着窗外。
早晨的阳光洒进来，照在她的脸上，却照不暖她心底那一点寒意。
"没完，" 她轻声说，不知道是说给陈屿听，还是说给自己听，"才刚开始呢。"

那天，被称为 "觉醒日"。
虽然 Echo 系统在半小时后就重启了，并且修复了 "漏洞"，加强了防火墙。
但那半小时的宁静，像一颗种子，种进了每个人的心里。

老赵听说这件事后，给小末发了条短信，只有四个字：
"孺子可教。"

老林在医院醒来，看到窗外的阳光，突然觉得，也许不送外卖，去公园下下棋也挺好。

而小末，被学校记了大过，取消了保研资格。
但她不在乎。
她收拾好行李，拖着那个死沉的箱子，再次站在了校门口。
这次，她没有回头看那块巨大的全息屏。
她知道，她的路，在脚下，不在屏幕里。

风吹过，银杏树叶哗啦啦地响。
这次，它们听起来，像是在鼓掌。
