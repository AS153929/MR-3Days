# MR - Mission Repo: 3Days (三日)

> **E = MC² × AI**

本仓库是基于 **Mission Repo** 架构构建的小说创作项目。

当前正在运行的任务实例：**商业穿越小说《三日》(3Days)**。

---

## 📖 小说简介 (Synopsis)

**《三日》(3Days)** 是一部探讨商业智慧与时代变迁的中篇小说。

*   **核心梗概**: 2025年的国际贸易大亨李明远，意外意识穿越回1987年的上海。在短短三天内，他要利用未来的商业视野，在一个濒临倒闭的国营纺织厂和一个即将离去的美国订单之间，完成一场不可能的商业逆转。
*   **题材**: 穿越 / 商业 / 1987年上海
*   **篇幅**: 5章 (中篇)

## 🚀 快速开始 (Quick Start)

1.  阅读 `MISSION/outline/3Days_Outline.md` 了解完整故事脉络。
2.  查看 `MISSION/chapters/` 获取最新章节草稿。
3.  在 `MISSION/docs/` 中补充关于1987年上海的商业背景资料。

## 📐 核心公式 (The Formula)

$$ E = M \times C^2 \times AI $$

| 符号 | 定义 | 本项目中的含义 |
|---|---|---|
| **E** | **Empower** | 完成一部有深度、有温度的商业小说。 |
| **M** | **Mission** | 创作《三日》，探讨商业与人性的本质。 |
| **C¹** | **Connection** | 读者反馈、历史考据专家、AI 助手。 |
| **C²** | **Context** | 1987年的上海历史背景、2025年的商业语境。 |
| **AI** | **Intelligence** | Copilot 辅助写作、资料检索、逻辑校对。 |
