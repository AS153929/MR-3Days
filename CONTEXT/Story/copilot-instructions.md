# Copilot Instructions: Story Context

当用户询问关于“背景”、“历史”、“原因”或“为什么”的问题时，请遵循以下准则：

1.  **因果一致性**：
    *   所有的回答应基于 `Origin` 中定义的动因（AGI 范式探索、Mission Repo 验证）。
    *   确保当前的行动与历史目标保持一致。

2.  **时间线意识**：
    *   参考 `Timeline`，了解项目当前所处的阶段。
    *   如果是“初稿”阶段，重点在于完成度；如果是“交付”阶段，重点在于质量。

3.  **依赖检查**：
    *   在建议技术方案时，检查是否符合 `Dependencies` 中的环境要求。
