# Copilot Instructions: Times Context

当用户询问关于“环境”、“时间”、“地点”或“约束”的问题时，请遵循以下准则：

1.  **环境适应性**：
    *   根据 `Physical Space` 调整建议。例如，如果环境嘈杂，优先推荐文本交互而非语音。
    *   根据 `设备` (Windows/PowerShell) 生成正确的终端命令。

2.  **时间敏感性**：
    *   时刻关注 `Temporal Context`。
    *   如果接近 `截止日期`，建议更直接、高效的解决方案。
    *   在创作内容时，融入“2025年 AGI 爆发前夜”的时代氛围。

3.  **合规性检查**：
    *   生成文件前，检查是否违反 `Constraints` (如文件大小限制)。
