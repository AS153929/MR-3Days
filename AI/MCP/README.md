# 🔌 MCP: Model Context Protocol

> AI > MCP

## 📖 简介
MCP (Model Context Protocol) 是连接 LLM 与外部数据/工具的标准协议。

## 🛠️ Recommended Servers (推荐配置)

### 1. Filesystem (本地文件系统)
- **用途**: 允许 AI 读取和修改本地文件（已默认集成）。
- **配置**: 无需额外配置。

### 2. GitHub (代码托管)
- **用途**: 允许 AI 读取 Issue, PR, 搜索代码库。
- **配置**: 需配置 GitHub Token。

### 3. Memory (记忆体)
- **用途**: 连接外部向量数据库或知识库。
- **状态**: *Planned (规划中)*
